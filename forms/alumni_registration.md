**Alumni Database Form**

**Instructions**

- Fill one form for each alumnus/alumna.
- Fields marked (\*) are mandatory.
- Use dropdown options wherever available.
- If the alumnus has participated in multiple Gyan Setu visits, add multiple entries in the visit section.
